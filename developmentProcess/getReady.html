# 确定电脑上是否安装了nodejs

如果没安装，从官网下载最新版本的node.js安装 打开cmd命令窗口，切换到要新建项目的文件夹下，可以使用 node –v 来查看node版本，出现版本号则node安装完成。

Node.js下载地址

# 全局安装vue-cli

在命令行输入：

npm install –g vue-cli （全局安装vue-cli）

vue init webpack my-project （创建一个基于webpack模板的项目，my-project为项目文件夹名称）

进入my-project文件夹下执行：

npm install	（安装依赖）

npm run dev	（启动项目）
